Метод розділення змінних (метод Фур'є) для гіперболічних рівнянь

Задача 1

Постановка задачі

Рівняння: Utt = 4Uxx
Граничні умови:
U(0, t) = 0
U(1, t) = 0
Початкові умови:
U(x, 0) = 0
Ut(x, 0) = 3 sin(πx)

Розв'язання методом розділення змінних

Крок 1 Розділення змінних:
U(x, t) = X(x) · T(t)
Підставляємо в рівняння:
X(x)T''(t) = 4X''(x)T(t)
Розділяємо змінні:
T''(t) / (4T(t)) = X''(x) / X(x) = -λ
Крок 2 Задача Штурма-Ліувілля для X(x):
X''(x) + λX(x) = 0
з граничними умовами: X(0) = 0, X(1) = 0

Власні значення: λn = (nπ)2, n = 1, 2, 3, ...
Власні функції: Xn(x) = sin(nπx)
Крок 3 Рівняння для T(t):
T''(t) + 4λnT(t) = 0
T''(t) + 4(nπ)2T(t) = 0
Загальний розв'язок:
Tn(t) = An cos(2nπt) + Bn sin(2nπt)
Крок 4 Загальний розв'язок:
U(x, t) = Σn=1∞ [An cos(2nπt) + Bn sin(2nπt)] sin(nπx)
Крок 5 Визначення коефіцієнтів з початкових умов:

З умови U(x, 0) = 0:
U(x, 0) = Σn=1∞ An sin(nπx) = 0
Отже, An = 0 для всіх n.

З умови Ut(x, 0) = 3 sin(πx):
Ut(x, 0) = Σn=1∞ 2nπBn sin(nπx) = 3 sin(πx)

Порівнюючи коефіцієнти:
2πB1 = 3 ⟹ B1 = 3/(2π)
Bn = 0 для n ≠ 1

Фінальний розв'язок:

U(x, t) = (3/(2π)) sin(2πt) sin(πx)
Задача 2

Постановка задачі

Рівняння: Utt = 9Uxx
Граничні умови:
U(0, t) = 0
U(1, t) = 0
Початкові умови:
U(x, 0) = cos(15πx)
Ut(x, 0) = 0

Розв'язання методом розділення змінних

Крок 1 Розділення змінних:
U(x, t) = X(x) · T(t)
Підставляємо в рівняння:
X(x)T''(t) = 9X''(x)T(t)
Розділяємо змінні:
T''(t) / (9T(t)) = X''(x) / X(x) = -λ
Крок 2 Задача Штурма-Ліувілля для X(x):
X''(x) + λX(x) = 0
з граничними умовами: X(0) = 0, X(1) = 0

Власні значення: λn = (nπ)2, n = 1, 2, 3, ...
Власні функції: Xn(x) = sin(nπx)
Крок 3 Рівняння для T(t):
T''(t) + 9λnT(t) = 0
T''(t) + 9(nπ)2T(t) = 0
Загальний розв'язок:
Tn(t) = An cos(3nπt) + Bn sin(3nπt)
Крок 4 Загальний розв'язок:
U(x, t) = Σn=1∞ [An cos(3nπt) + Bn sin(3nπt)] sin(nπx)
Крок 5 Визначення коефіцієнтів з початкових умов:

З умови Ut(x, 0) = 0:
Ut(x, 0) = Σn=1∞ 3nπBn sin(nπx) = 0
Отже, Bn = 0 для всіх n.

З умови U(x, 0) = cos(15πx):
U(x, 0) = Σn=1∞ An sin(nπx) = cos(15πx)

Розкладаємо cos(15πx) в ряд Фур'є по синусах на інтервалі [0,1]:
cos(15πx) = Σn=1∞ an sin(nπx)

Коефіцієнти Фур'є:
an = 2∫01 cos(15πx) sin(nπx) dx

Використовуючи формулу:
∫ cos(αx) sin(βx) dx = (1/2)[sin((α+β)x)/(α+β) - sin((α-β)x)/(α-β)]

Для n = 15:
a15 = 2∫01 cos(15πx) sin(15πx) dx = 2 · (1/2)∫01 sin(30πx) dx = 0

Для n ≠ 15:
an = 2 · [sin((15+n)π)/((15+n)π) - sin((15-n)π)/((15-n)π)]

Оскільки sin(kπ) = 0 для цілих k, отримуємо:
an = 0 для всіх n

Альтернативний підхід: Використовуємо формулу Ейлера та тотожність:
cos(15πx) = (ei15πx + e-i15πx)/2

Або використовуємо розклад через косинуси:
cos(15πx) = Σn=0∞ cn cos(nπx)

Оскільки cos(15πx) не є непарною функцією, а наш базис складається з sin(nπx), потрібно знайти коефіцієнти розкладу. Використовуючи інтеграл:
An = 2∫01 cos(15πx) sin(nπx) dx

Після обчислення (використовуючи формулу добутку тригонометричних функцій):
An = (2nπ/(n²π² - 225π²))[1 - (-1)ncos(15π)]

Оскільки cos(15π) = cos(π) = -1 (15 - непарне число), то:
An = (2nπ/(n²π² - 225π²))[1 - (-1)n(-1)] = (2nπ/(n²π² - 225π²))[1 + (-1)n]

Це дає ненульові коефіцієнти тільки для парних n. Однак, більш простий підхід:

Використовуючи тотожність sin(α)cos(β) = ½[sin(α+β) + sin(α-β)]:
An = 2∫01 cos(15πx) sin(nπx) dx = ∫01 [sin((n+15)πx) + sin((n-15)πx)] dx

Для n ≠ 15:
An = [-cos((n+15)πx)/((n+15)π) - cos((n-15)πx)/((n-15)π)]01
= [-cos((n+15)π)/((n+15)π) - cos((n-15)π)/((n-15)π)] + [1/((n+15)π) + 1/((n-15)π)]

Спрощуючи та враховуючи, що cos(kπ) = (-1)k:
An = [1 - (-1)n+15]/((n+15)π) + [1 - (-1)n-15]/((n-15)π)

Оскільки 15 - непарне, то (-1)n+15 = -(-1)n та (-1)n-15 = -(-1)n:
An = [1 + (-1)n]/((n+15)π) + [1 + (-1)n]/((n-15)π)
= [1 + (-1)n][1/((n+15)π) + 1/((n-15)π)]
= [1 + (-1)n] · (2nπ)/((n²-225)π²)
= 2n[1 + (-1)n]/((n²-225)π)

Фінальний розв'язок:

U(x, t) = Σn=1∞ An cos(3nπt) sin(nπx)
де An = 2n[1 + (-1)n]/((n²-225)π) для n ≠ 15
A15 = 0

Примітка: Коефіцієнти An відмінні від нуля тільки для парних n.

Пояснення методу

Метод розділення змінних (метод Фур'є) для гіперболічних рівнянь полягає в наступному:

  1. Шукаємо розв'язок у вигляді добутку функцій, що залежать тільки від однієї змінної
  2. Підставляємо в рівняння та розділяємо змінні
  3. Отримуємо задачу Штурма-Ліувілля для просторової частини
  4. Знаходимо власні значення та власні функції
  5. Розв'язуємо звичайне диференційне рівняння для часової частини
  6. Складаємо загальний розв'язок у вигляді ряду
  7. Визначаємо коефіцієнти з початкових умов, використовуючи розклад у ряд Фур'є
